# Get Started
1. clone project -> `cd <you project>`
2. create virtual env
```python
uv venv <your env name>
```
3. install dependencies
```python
uv pip install -r requirement.txt
# or
pip install -r requirement.txt
```
4. Run service
```python
fastapi dev <your server file>
```
